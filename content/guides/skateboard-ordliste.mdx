## Skateboard-ordliste — start her

Skate-miljøet har sit eget sprog. Denne ordliste forklarer **40+ begreber** på dansk — så du kan følge med i parken, på YouTube og i vores [guides](/guides).

**Helt ny?** Start med [sådan skubber du](/guides/saadan-skubber-du-paa-skateboard), [ollie](/guides/hvordan-laerer-man-ollie) og [skatepark-etikette](/guides/skatepark-etikette-danmark).

## Board og dele

| Ord | Betydning |
|-----|-----------|
| **Deck** | Træbrættet du står på |
| **Trucks** | Metal-delen under decket — holder hjul |
| **Hjul (wheels)** | Polyurethan-hjul — hårdhed måles i A (fx 99A) |
| **Bearings / kuglelejer** | Inde i hjul — får dem til at rulle |
| **Griptape** | Sandpapir-lignende overflade på deck |
| **Nose / tail** | For- og bagende af deck — tail bruges til pop |
| **Bolts / hardware** | Skruer der holder trucks på deck |
| **Complete** | Færdigbygget board — klar til kørsel |
| **Custom** | Deck + trucks + hjul samlet selv |
| **Popsicle deck** | Symmetrisk «klassisk» skateboard-form |

Guides: [complete vs. custom](/guides/complete-vs-custom-skateboard), [trucks](/guides/bedste-skateboard-trucks), [hjul](/guides/bedste-skateboard-hjul), [griptape](/guides/bedste-griptape-skateboard).

## Stance og kørsel

| Ord | Betydning |
|-----|-----------|
| **Regular** | Venstre fod forrest (typisk højrehåndet) |
| **Goofy** | Højre fod forrest |
| **Switch** | Kør i modsat stance |
| **Fakie** | Baglæns med normal stance — se [fakie-guide](/guides/saadan-koerer-du-fakie) |
| **Nollie** | Ollie med forfod-pop |
| **Push** | Skubbe dig frem — [guide](/guides/saadan-skubber-du-paa-skateboard) |
| **Mongo** | Skubbe med forfod (undgås af de fleste) |
| **Carve** | Svinge med vægt på kanter |
| **Pump** | Skabe fart i transition uden at skubbe |

## Ofte stillede spørgsmål

**Er ordene på engelsk i parken?**  
Ja — de fleste trick-navne er engelske worldwide. Denne liste oversætter til dansk.

**Hvad betyder «steez»?**  
Stil og attitude i kørsel — ikke et trick.

**Er der flere ord?**  
Ja — hundredvis. Denne liste dækker det du møder som begynder i Danmark.
